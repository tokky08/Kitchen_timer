# Kitchen_timer

ステータスバーに表示されるキッチンタイマーです。

ステータスバーとは最上部にあるWi-Fiや充電マークがあることろです。


## 開発ツール・ライブラリ

python 3.8

pyobjc-core==6.2

pyobjc-framework-Cocoa==6.2

rumps==0.3.0


## 使用目的
Macのステータスバーにキッチンタイマーを表示させたくて探したが好みのものがなかったので作ってみた。

開発中です。

## 使用方法

```bash
$ pip install -r requirements.txt
```

```bash
$ python timer.py
```

以上！
